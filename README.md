# Ktirio AI — Archviz + AI Rendering Knowledge Base

Turn your AI agent into an architectural visualization specialist. This skill gives your agent expert knowledge about generating photorealistic renders with AI — prompt libraries, workflow guides, best practices, and a complete archviz glossary.

## Install

```bash
npx skills add davidbenal/ktirio-ai-skill
```

Works with: Claude Code, Cursor, Codex, Gemini CLI, Windsurf, Kiro, and 7+ more agents.

## What's inside

| File | Content |
|------|---------|
| **SKILL.md** | Overview, quick start, navigation |
| **prompts.md** | Ready-to-use prompt templates — interiors, exteriors, landscape, aerial, commercial |
| **workflows.md** | Step-by-step guides for SketchUp, Revit, ArchiCAD, Blender, D5 Render |
| **best-practices.md** | Composition, camera angles, lighting, materials, image preparation checklist |
| **glossary.md** | 150+ archviz terms mapped to prompt keywords — materials, lighting, styles, vegetation |

## Example usage

After installing, ask your agent:

- "Write a prompt for a luxury modern kitchen render"
- "What's the best workflow from SketchUp to AI render?"
- "Help me describe this material for an archviz prompt"
- "What camera settings should I use for an interior render?"
- "Compare lighting conditions for a facade render"
